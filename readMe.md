
## Run Locally

Clone the project

```bash
  git clone https://github.com/nonesubham/Ztube-music.git
```

Go to the project directory

```bash
  cd ztube music
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
#### Note: This app is under development so, crashes and lag possible to face.
